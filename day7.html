Web Development?

Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites. The word Web Development is made up of two words, that is:
Web: refers to websites, web pages or anything that works over the internet. Development: It refers to building the application from scratch.
>>Web Development can be classified into two ways:
1.Frontend Development
2.Backend Development

To know more about Web Development


1.Frontend Development

Front-end development is the process of building components that interact with users. Examples are the user interface, buttons, user-entered data, websites, and user experience (UX) features.
